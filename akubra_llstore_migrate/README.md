# Akubra Llstore Migration Utility

Run from your CATALINA_HOME:


```
Usage: akubra_llstore_migrate.rb [options]

options:

    -f, --from                 current akubra_llstore configuration
    -t, --to                   akubra_llstore configuration to migrate to
    -o, --objectStore
    -d, --datastreamStore
    -h, --help
```
